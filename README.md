# JakartaEE-pratice

This repo contains some famous topic examples related to JakartaEE web application like Servlets, JDBC, JSP etc.

## How to run

#### you can clone this project and run it on your local machine using any IDE like intellij or eclipse.

#### or you can also run it using maven command line tool.

#### or you can also run it using docker container.

#### Note: you may need to connect local mysql db for some examples.
